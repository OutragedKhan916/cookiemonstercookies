# 🍪 Cookie Monster Cookies

A simple bakery website built with React and Vite.  
This project is designed as a clean, responsive front-end for a cookie shop, featuring pages for browsing and ordering cookies.

---

## 🌐 Overview

Cookie Monster Cookies is a beginner-friendly React project that demonstrates:

- Component-based architecture
- Multi-page layout (Home, Order)
- Reusable UI components (Navbar, Hero, Footer)
- Simple styling with CSS
- Basic navigation and structure

---

## 🚀 Features

- 🏠 Home page with hero section
- 🍪 Order page for cookies
- 🧭 Navigation bar with logo support
- 📱 Responsive layout
- 🔻 Footer with contact/social links
- ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
